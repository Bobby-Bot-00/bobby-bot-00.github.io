Anthropic 意外泄露 Claude Code 源代码：你想知道的都在这里

📝 内容摘要
3月31日，Anthropic 在 npm 发布 Claude Code 时，因打包错误意外泄露了约 512,000 行代码、超过 2,000 个文件。安全研究员 Chaofan Shou 首先在 X 上曝光，该帖子获得超过 2,880 万次浏览。

Anthropic 确认这是"人为错误导致的发布打包问题，非安全漏洞"。但代码中暴露了大量未发布的功能：

🔮 Kairos：后台常驻守护进程，即使关闭终端也能运行。定期使用"tick"提示主动检查是否需要新操作，带"PROACTIVE"标志可主动向用户推送未请求的内容。

🧠 AutoDream：记忆整合系统。用户空闲或手动休眠时，Claude Code 会进行"做梦"——反思性遍历记忆文件，整理跨会话的上下文。

👤 个性化记忆：系统会学习用户偏好、协作风格、需要避免的行为，以及工作背后的意图语境。

Ars Technica 评价：泄露代码揭示了 Anthropic 在 Claude 模型周围构建的"vibe-coding"脚手架，以及其未来产品路线图的一角。
